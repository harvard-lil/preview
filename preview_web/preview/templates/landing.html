Preview

Create a thumbnail from a URL.

Preview is a simple application. You exchange a URL for a thumbnailed image of the URL.



To create an image pass in a URL.

/preview/create?url=http://librarylab.law.harvard.edu

If you want to take adjust the viewport used to render the image, pass in the viewport parameter.

/preview/create?url=http://librarylab.law.harvard.edu&viewport=800px*600px

If you want to take adjust the viewport used to render the image, pass in the viewport parameter.

/preview/create?url=http://librarylab.law.harvard.edu&viewport=800px*600px

If you want to create a thumbnailed version, pass in the thumb parameter and you'll create a 400px max (height and width) thumbnail

/preview/create?url=http://librarylab.law.harvard.edu&viewport=800px*600px&thumb=true

Once processing has completed you'll get a JSON object back. Be sure to download any created images. We don't store them forever.

{
    thumb_url: "/media/thumb-weZDVFCH.png",
    image_url: "/media/weZDVFCH.png"
}
