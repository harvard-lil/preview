Preview

Create an image from a URL.



To create an image pass in a URL.

/preview/create?url=http://librarylab.law.harvard.edu

If you want to create a thumbnailed version, pass in the thumb parameter with the max width of the thumbnail. This'll create a thumbnail with a fixed width and a variable length.

/preview/create?url=http://librarylab.law.harvard.edu&thumb=400px

If you want to the thumbnail to be a fixed width and height, have Preview slice it for you.

/preview/create?url=http://librarylab.law.harvard.edu&thumb=400px*300px

If you want to adjust the viewport used to render the image, pass in the viewport parameter.

/preview/create?url=http://librarylab.law.harvard.edu&thumb=400px&viewport=800px*2000px

If you're trying to generate a preview for a URL that contains things like ampersands, be sure to URL encode the URL first.



Once processing has completed you'll get a JSON object back. Be sure to download any created images. We don't store them forever.

{
    thumb_url: "/preview/media/thumb-weZDVFCH.png",
    image_url: "/preview/media/weZDVFCH.png"
}
