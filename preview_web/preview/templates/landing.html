Preview

Create a thumbnail from a URL.


Preview is a simple application. You exchange a URL for a thumbnailed image of the URL.



To create an image, call preview like this,

/preview/create?url=http://librarylab.law.harvard.edu


If you want to take just a slice of that image pass in slice-size parameter,

/preview/create?url=http://librarylab.law.harvard.edu&slice-size=800px*600px
